# База знаний IT-поддержки

## 1. CI/CD и DevOps

### Jenkins Pipeline
**Проблема:** npm ERR! EACCES при сборке
**Решение:**
- Проверить права доступа к папке node_modules
- Выполнить: `sudo chown -R $(whoami) ~/.npm`
- Очистить кэш: `npm cache clean --force`
- Переустановить зависимости: `rm -rf node_modules && npm install`

### Docker
**Проблема:** ImagePullBackOff
**Решение:**
- Проверить доступность Docker registry
- Выполнить: `docker system prune -a`
- Перезапустить Docker daemon
- Проверить сетевые настройки

**Проблема:** Conflict. The container name is already in use
**Решение:**
- Удалить старый контейнер: `docker rm -f container_name`
- Или использовать другой имя при запуске

### GitLab CI/CD
**Проблема:** Runner не подключается
**Решение:**
- Проверить токен runner'а
- Обновить конфигурацию .gitlab-ci.yml
- Перезапустить runner: `gitlab-runner restart`

### Kubernetes
**Проблема:** Helm timeout, pod connection refused
**Решение:**
- Проверить ConfigMap версии
- Увеличить timeout в values.yaml
- Проверить сетевые политики
- Выполнить: `kubectl describe pod pod_name`
